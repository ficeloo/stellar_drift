# GDD — Stellar Drift

## La vision
Stellar Drift est un jeu volontairement minimaliste reprenant le concept du jeu Asteroids d'Atari, où le but est de nettoyer des vagues successives d'astéroïdes et de survivre le plus loin possible.

On évolue ici dans un petit jeu avec pour architecture l'Entity Component System. La boucle de gameplay est simple mais nerveuse.

Pour ce projet, j'ai décidé de garder une architecture minimaliste afin de me concentrer sur les parties importantes de l'ECS.
